# Location Map Quick Start Guide

## What's New?

Quick Log now includes a Location Map feature that lets you visualize your logged locations on an interactive map!

## Accessing the Location Map

### From Main Screen
1. Tap the **location icon** in the toolbar
2. The map opens showing all your logged locations

### From Entries Overview
1. Go to Entries Overview
2. Tap the **location icon** in the toolbar
3. View your location history on the map

## Using the Map

### View Locations
- **Markers** show each logged location
- **Tap a marker** to see details (location name, date, tags)
- **Pinch/zoom** to explore the map
- **Pan** by dragging

### Filter by Date
1. Tap **"Filter Dates"** button
2. Select start and end dates from the calendar
3. Tap **OK** to apply filter
4. Map updates to show only entries in that date range
5. Tap **"Clear"** to remove the filter

### View Timeline
1. Tap **"Timeline"** button
2. See a chronological list of your visits
3. Each entry shows:
   - Date and time
   - Location name
   - Associated tags
4. Tap **"Share"** to export the timeline

## Troubleshooting

**Q: Map is blank?**
- Check internet connection (needed for map tiles)
- Ensure location permissions are granted
- Make sure you have logged entries with location data

**Q: Some locations missing?**
- Check if a date filter is active
- Entries without coordinates won't show on map
- Clear filter to see all entries

**Q: Can't see location labels?**
- Some entries may only have coordinates
- This happens if geocoding wasn't available
- Tap marker to see coordinates
